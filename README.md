This is a very simple version of the popular game Battleship, the objective of the game is to find 5 ships on a 10x10 grid, one that is 5 units long, one that is 4 units, two that are 3 units and one that is 22 untis long. You play the game by inputting a row and a column where you "Shoot" to try and find a ship. This game features a CPU opponent that is trying to sink the ships aswell (THE CPU HAS NO ACTUAL INPUTS OR EFFECT ON THE BOARD SEEN BY THE PLAYER) and you race the CPU to find the ships, this game features a randomizing board which will allow for the game to be played several times each with a different expeirience, Hope you enjoy!
